# fbapicli

Command line interface for Facebook API.

## Usage

```shell
$ FB_ACCESS_TOKEN="YOUR_ACCESS_TOKEN" ./fbapicli sample.toml
```

### Environment Variables

* `FB_ACCESS_TOKEN`(required)
  * Your access token
* `FB_DEBUG`(optional)
  * Enable debug mode
